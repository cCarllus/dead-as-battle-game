Game Notes

Updates, patch notes and developer announcements.

New Features

  • Team/party system integrated with global chat invite flow.
  • Invite popup with accept/decline, team panel controls and leader permissions.
  • Global online presence counter synced from the websocket room.
  • Lobby Spotify player rendered only on Home screen.

Gameplay & UX

  • Champion preview scaled and framed to prioritize waist-up visibility.
  • Team area moved above Quick Match controls for better flow.
  • Chat invite action now shows check state for members already in your team.
  • Dedicated K/D card for selected champion with clear icon indicators.

Known Issues

  • Team state is in-memory only and resets when the server restarts.
  • Spotify autoplay may require first user interaction depending on browser policy.

New Heroes

  • Sukuna added to the roster with high-pressure damage profile.
  • New ultimate attack focused on burst damage against unshielded targets.

Gameplay Changes

  • Improved server-side hit detection for projectile-based weapons.
  • Strafe acceleration reduced by 5% for more readable tracking duels.

Bug Fixes

  • Fixed reload animation freeze when interrupted by dash actions.
  • Resolved collision issues blocking map doorway traversal.
